🌐 Portfolio Website — Tech Stack & Implementation
Built as a single-file HTML5/CSS3/JavaScript portfolio with zero external frameworks or build tools — just pure vanilla web technologies running directly in the browser.
The navbar uses a floating pill-shaped layout with backdrop-filter: blur(28px) saturate(180%) layered with CSS gradient overlays and inset highlight borders to achieve the liquid glass morphism effect. Theme switching between dark and light mode is handled via CSS custom properties (data-theme attribute on the root element) with localStorage persistence.
The hero section combines a CSS radial-gradient mesh background, animated floating orbs using @keyframes, a dot-grid pattern with SVG mask-image, and a glassmorphism profile card with a gradient avatar ring. The typewriter name animation is built in plain JavaScript using a recursive setTimeout loop with randomized character delay for a natural feel.
The resume section renders fully inside the page as a two-column layout — a gradient sidebar with animated skill bars and a structured main panel with education, projects, experience, and certifications. Skill bars animate on scroll using the IntersectionObserver API. The download button triggers window.print() for a clean PDF export.
The AI chatbot is a floating widget built with CSS transitions (opacity + transform: scale) for smooth open/close. It uses a keyword-based pattern matching engine in JavaScript (RegExp.test()) to match user queries against predefined response categories — covering skills, projects, education, experience, certifications, and availability — with a simulated typing delay for realism.
Scroll reveal animations across all sections use a single shared IntersectionObserver instance that adds a .vis class triggering CSS opacity and translateY transitions. All styling uses CSS custom properties for full theme consistency across every component.
